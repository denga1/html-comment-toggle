## [1.0.0] - 2025-10-29

### 新增
- 初始版本发布
- 支持在 Vue 文件的 template 部分进行 HTML 注释切换
- 智能处理内部注释转义（避免嵌套注释问题）
- 添加注释：内部 `<!--` → `<!-*`，`-->` → `*->`
- 取消注释：内部 `<!-*` → `<!--`，`*->` → `-->`
