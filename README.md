# HTML Comment Toggle

HTML 注释切换工具，解决多行注释的问题。

## 核心特性

**智能处理内部注释**：由于 HTML 不支持嵌套注释，本插件会将选中代码内的注释进行转义处理。

- 添加注释：内部 `<!--` → `<!-*`，`-->` → `*->`
- 取消注释：内部 `<!-*` → `<!--`，`*->` → `-->`

## 使用方法

1. 选中要切换注释的代码
2. 按 `Ctrl+Shift+/` (Windows/Linux) 或 `Cmd+Shift+/` (Mac)
3. 或在命令面板搜索 "HTML Comment Toggle"
